# Pourō-Claude

**Pour slowly. Brew deeply.**

Pourō は、ハンドドリップコーヒーの抽出を迷わず進めるための、静かなブリューガイド＆タイマーPWAです。

設定 → レシピ確認 → 抽出タイマー → 記録 → 振り返りまでを、ひとつの画面体験としてまとめています。

## 概要

Pourō は、ハンドドリップ中に「次に何をすればよいか」を分かりやすく表示するためのPWAです。

主な目的は以下です。

- 抽出手順を迷わず進める
- 注湯量・累計湯量・時間を確認しやすくする
- 抽出後の記録を残し、次回の調整につなげる
- iPhoneのホーム画面に追加して、アプリのように使う

---

## 対応メソッド

現在のバージョンでは、以下の4つの抽出メソッドに対応しています。

- 4:6 Method
- Hybrid
- THE NEO BREW
- Ice Brew

各メソッドは、Pourō内でレシピ設定・プレビュー・抽出タイマー・抽出ログに反映されます。

---

## 主な機能

### Brew

抽出メソッドを選択し、レシピ設定へ進む画面です。

- 4メソッドの選択
- メソッド説明の表示
- Recipe Setupへの導線

### Recipe Setup

抽出条件を設定する画面です。

- 豆量の選択
- 比率の調整
- 4:6 Method用の味・濃度設定
- レシピ概要の確認

### Preview

抽出前に、注湯タイムラインを確認する画面です。

- Coffee / Water / Ratio
- Pours / Drawdown
- 注湯ステップ一覧
- 抽出開始ボタン

### Active Brew

抽出中に使うタイマー画面です。

- 経過時間
- 現在の注湯量
- 累計湯量
- 現在ステップの指示
- 次ステップ予告
- Pause / Back / Next 操作

### Brew Log

抽出後の記録画面です。

- 評価
- 味タグ
- 次回メモ
- 豆・挽き目
- ドリッパー
- 湯温
- 履歴保存

### History

保存した抽出記録を確認する画面です。

- 抽出履歴一覧
- 評価表示
- メモ表示
- 器具情報表示
- 総記録数・平均評価

### Settings

アプリの状態や保存データを確認する画面です。

- Default brew表示
- Brew assist表示
- History logs件数
- Clear history
- About表示

## PWAとしての使い方

Pourō はPWAとして動作します。

### iPhoneでホーム画面に追加する方法

1. Safariで公開URLを開く
2. 共有ボタンを押す
3. 「ホーム画面に追加」を選択
4. 名前を確認して追加
5. ホーム画面のPourōアイコンから起動

ホーム画面から起動すると、通常のWebページではなく、アプリに近い表示で使用できます。
